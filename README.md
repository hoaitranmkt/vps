# Install New VPS

🔷 Install Nginx + Nginx UI

```bash
wget https://raw.githubusercontent.com/hoaitranmkt/vps/refs/heads/main/nginx.sh -O nginx.sh \
  && chmod +x nginx.sh \
  && sudo ./nginx.sh
```
